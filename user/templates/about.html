This website is used for the collection and sharing of web links and can be operated online through any web browser.Thus, cross-platform and independent of specific browser. It is also convenient to share your bookmarks with others.

Advantage
  • Can be shared and visible to anyone
  • Free
  • Simple, support uploading existing (i.e. html format file exported from browser) bookmarks
  • connectionless, username can be random letters. If not provided, no contact information, including the webmaster
Disadvantage
  • Everything in a bookmark is public and can be seen by others.
  • Because it is stored over the network, the stability is not as good as the browser's own bookmark, and server or network problems may cause the content to fail to save remedy
  • Need to log in, certain amount of time to consume; if just visit bookmarks without edition, you can visit the shared address without signing in
  • For bookmark visitors( not the owner ), there are ads
