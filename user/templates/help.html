用法

添加、修改、删除书签或文件夹

在对应的书签图标或文件夹名称上用鼠标点击，弹出快捷菜单，选择对应的操作

访问对应的书签

点击书签，网页将在新的标签页中打开

展开、折叠文件夹

显示或者隐藏该文件夹的子文件夹和子书签

名词释义

同步至服务器

将修改后的书签同步到服务器，这样更新的内容可以保留，下次访问的时候可以访问更新后的内容，若改变书签后不进行同步，新的内容将丢失

退出

退出登录，你若需要再次访问该页面，需要重新登录；同时他人也无法使用您的浏览器对您的书签进行修改

清除账户

将您的书签及账户名从本网站服务器中删除，如果您需要这些书签，请保存到本地。方法如下：

  1. 登录后，浏览器的地址栏中显示URL，例如：http://127.0.0.1:8000/owner.html?2709821b9389377c472f64e7981ffa4894a78b4dd0185999543857c11fdce3b8
  2. 在浏览器地址栏中将“owner.html?”替换为“c/”，上述的地址将变为：http://127.0.0.1:8000/c/2709821b9389377c472f64e7981ffa4894a78b4dd0185999543857c11fdce3b8
  3. 按下Enter键
  4. 此时，浏览器中的内容就是你的书签
  5. 将内容复制到文本文件中并保存，以免丢失
上传文件

当您想把已有的(从浏览器中导出的，html格式的)书签变成在线书签时，可以将书签文件上传。或者服务器崩溃，您保存了书签到本地，现在可以重新上传。注意，它将覆盖已有的书签，而不是将其添加到已有的书签中。

Usage

Add, modify, delete bookmarks or folders

Click on the corresponding bookmark icon or folder name, pop up the shortcut menu, select the corresponding action

Access the corresponding bookmark

Click bookmark and the page will be opened in the new tab

Expand & collapse

Display or hide it's sub-folder and bookmark

Interpretation

Sync to Server

Sync the modified bookmark to the server, so that the updated content will be saved. if the bookmark is not synced after change, the new content will be lost.

Log Out

Remove "login" state, you need to log in again if you want to edit later. If you need to forbid others to edit your bookmark with this browser, it helps.

Remove account

Remove all your information from this website, non-recoverable. If the bookmark matters, save it to local file.Step as follows:

  1. After Login,the web browser will show URL in location bar. For example,http://127.0.0.1:8000/owner.html?2709821b9389377c472f64e7981ffa4894a78b4dd0185999543857c11fdce3b8
  2. Replace "owner.html?" with "c/"，above URL will be http://127.0.0.1:8000/c/2709821b9389377c472f64e7981ffa4894a78b4dd0185999543857c11fdce3b8
  3. Press Enter
  4. Now, your bookmark is shown
  5. Copy All content and paste it in a file, save the file
Upload File

Upload an existing bookmark file ( which is exported from your browser, html format ), it will be an online bookmark.Or when server crashed, you stored a local bookmark, you can upload it now. Caution, it will overwrite existing bookmark，instead of adding them to existing one.
