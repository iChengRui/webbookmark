用法

添加、修改、删除书签或文件夹

在对应的书签图标或文件夹名称上用鼠标点击，弹出快捷菜单，选择对应的操作

访问对应的书签

将鼠标移动到想要访问的网址上，按下鼠标滑轮，网页将在新的标签页中打开

名词释义

同步至服务器

将修改后的书签同步到服务器，这样更新的内容可以保留，下次访问的时候可以访问更新后的内容，若改变书签后不进行同步，新的内容将丢失

退出

退出登录，你若需要再次访问该页面，需要重新登录;同时他人也无法使用您的浏览器对您的书签进行修改

清除账户

将您的书签及账户名从本网站服务器中删除，如果您需要这些书签，请注意先保存到本地

上传文件

当您想把自己已有的(从浏览器中导出的)书签上传，变成在线书签时，可以使用此功能。注意"将覆盖已有的书签",而不是将其添加到已有的书签中。或者当本服务器崩溃，导致无法上传时，仅限在PC端的浏览器，您可以保存书签到本地，需2-3分钟，等待后续上传。

网站图标不可见或与实际不符

本网站直接从对应的网站获取图标(favicon.ico),若该网站不提供或无法通过访问获得，则会被替代或不可见

Usage

Add, modify, delete bookmarks or folders

Click with the mouse on the corresponding bookmark icon or folder name, pop up the shortcut menu, select the corresponding action

Access the corresponding bookmark

Move the mouse over the URL you want to access, press the mouse pulley, and the page will open in the new tab

Noun Interpretation

Sync to serve

Sync the modified bookmark to the server so that the updated content can be retained, the updated content can be accessed the next time accessed, and if the bookmark is not synced after the change, the new content will be lost

Exit

Sign in, and if you need to visit the page again, you need to log back in, and no one else will be able to use your browser to modify your bookmarks

Clear your account

Remove your bookmarks and account names from the servers of this website, if you need them, please note that save them locally

Upload a file

You can use this feature when you want to upload an existing (exported) bookmark from your browser and turn it into an online bookmark. Note that "will overwrite existing bookmarks" instead of adding them to existing bookmarks. Or when the server crashes, resulting in an inability to upload, only in the PC browser, you can save the bookmark to the local, it takes 2-3 minutes, waiting for subsequent uploads.
